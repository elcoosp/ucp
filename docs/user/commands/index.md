# CLI Command Reference

| Command | Description |
|---------|-------------|
| [`bootstrap`](bootstrap.md) | Extract a spec from source code |
| [`merge`](merge.md) | Merge multiple specs |
| [`curate`](curate.md) | Interactive conflict resolution |
| [`diff`](diff.md) | Structural spec comparison |
| [`verify`](verify.md) | Drift detection against source |
| [`registry`](registry.md) | Local spec store |
| [`export-all`](export-all.md) | Export to all formats |
| [`watch`](watch.md) | File‑watch continuous rebuild |
| `generate` | Generate framework code |
| `validate` | Check spec for conflicts |
| `components` | List/inspect components |
| `export` | Export to a single format |
| `import` | Import from external formats |
| `dashboard` | Generate HTML dashboard |
| `contract` | Export AI contract |
| `mcp` | Run MCP server |
