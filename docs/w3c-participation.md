## Status

- **Group:** UI Specification Schema Community Group

- **Representative:** UCP Team
- **Reference Implementation:** `ucp-spec.w3c.json` (exported via `ucp export --target w3c` or `ucp bootstrap --w3c`)

## Mission Alignment

The Community Group's mission — to define a "common, implementation-agnostic meta-model
for specifying the design, layout, behaviour, and constraints of user interface elements" —
directly aligns with UCP's Canonical Abstract Model (CAM).

UCP provides:
- A working extraction pipeline from real-world component libraries (Dioxus, Leptos, React).
- A fully populated schema with component anatomy, states, variants, properties, events.
- Integration with design tokens (DTCG) and accessibility metadata.
- A reference implementation that demonstrates the schema in practice.
