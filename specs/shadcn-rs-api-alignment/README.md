# Shadcn UI for Rust – API Alignment Specification Suite

This directory contains the complete specification suite for aligning the public APIs of Leptos‑shadcn/ui and Dioxus‑shadcn/ui under a common, framework‑agnostic contract.

## Documents

| File | Purpose |
|------|---------|
| `shadcn-rs-api-alignment-vision.md` | Product vision and strategic alignment |
| `shadcn-rs-api-alignment-brs.md` | Business & Stakeholder Requirements |
| `shadcn-rs-api-alignment-srs.md` | Software Requirements Specification |
| `shadcn-rs-api-alignment-architecture.md` | Architecture & Design Specification |
| `shadcn-rs-api-alignment-test.md` | Behavioral Specification & Test Verification Plan |

These documents are versioned and should be updated as the alignment effort progresses.
